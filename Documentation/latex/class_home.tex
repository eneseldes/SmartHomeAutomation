\doxysubsection{Detailed Description}
Singleton class that represents a home with users and rooms. 

The \doxylink{class_home}{Home} class follows the Singleton pattern to ensure only one instance of the home exists. It contains information about users and rooms in the home, and methods for saving and loading the state of the home.
