\doxysubsection{Detailed Description}
A class representing a motion detector that triggers an alarm when motion is detected. 

The \doxylink{class_motion_detector}{Motion\+Detector} class is a specialized type of detector that checks for motion in a room. If motion is detected, it triggers the associated alarm. This class inherits from the \doxylink{class_detector}{Detector} class and implements its abstract methods.
