\doxysubsection{Detailed Description}
Represents a device with a configurable attribute. 

\doxylink{class_settable_device}{Settable\+Device} is a base class for devices that have an attribute that can be adjusted within a specified range. This attribute can be modified using setter functions, and the bounds for the attribute are defined by lower and upper limits.
