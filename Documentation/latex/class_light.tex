\doxysubsection{Detailed Description}
A class representing a light device. 

The \doxylink{class_light}{Light} class inherits from the \doxylink{class_device}{Device} class and provides implementation for abstract methods. It represents a light device, including its brightness level and energy consumption.

The \doxylink{class_light}{Light} class inherits from the \doxylink{class_settable_device}{Settable\+Device} class and implements abstract methods to represent a light device, including its energy consumption, thread level, and display functionality.
