\doxysubsection{Detailed Description}
Represents a security detector that is linked to a \doxylink{class_security_alarm}{Security\+Alarm}. 

Abstract class representing a security detector that is linked to a \doxylink{class_security_alarm}{Security\+Alarm}.

This class inherits from the \doxylink{class_security_device}{Security\+Device} class and is associated with an instance of the \doxylink{class_security_alarm}{Security\+Alarm} class.

This class inherits from the \doxylink{class_security_device}{Security\+Device} class and is associated with an instance of the \doxylink{class_security_alarm}{Security\+Alarm}. It serves as a base class for different types of security detectors that perform detection and trigger alarms.
