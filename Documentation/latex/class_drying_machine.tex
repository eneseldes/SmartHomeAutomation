\doxysubsection{Detailed Description}
Represents a \doxylink{class_drying_machine}{Drying\+Machine} device with various drying modes and corresponding timers. 

Represents a drying machine that can operate in different modes.

The \doxylink{class_drying_machine}{Drying\+Machine} class inherits from \doxylink{class_has_mode}{Has\+Mode} and allows the user to select different drying modes such as "{}\+Normal"{}, "{}\+Delicate"{}, "{}\+Baby"{}, "{}\+Quick"{}, "{}\+Synthetic"{}, and "{}\+Anti\+Wrinkle"{}. Each mode has a corresponding timer setting. The class provides methods to display information, set modes, and calculate energy consumption.

The \doxylink{class_drying_machine}{Drying\+Machine} class inherits from \doxylink{class_has_mode}{Has\+Mode} and provides functionality to set various drying modes. Each mode has a predefined time and affects the energy consumption and thread level of the machine.
