\doxysubsection{Detailed Description}
Represents a general device with a name and an active status. 

This class provides basic functionality for managing devices such as opening, closing, checking if they are active, and comparing devices by their names.

This class provides basic functionality for managing devices such as opening, closing, checking if they are active, and comparing devices by their names. Derived classes should implement specific behavior for displaying the device, calculating energy consumption, and determining thread levels.
