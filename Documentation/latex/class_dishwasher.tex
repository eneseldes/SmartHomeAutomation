\doxysubsection{Detailed Description}
Represents a \doxylink{class_dishwasher}{Dishwasher} device with adjustable modes and timer functionality. 

The \doxylink{class_dishwasher}{Dishwasher} class inherits from \doxylink{class_has_mode}{Has\+Mode} and provides functionality to set and retrieve operation modes, display information about the device, and calculate energy consumption. It includes predefined modes like "{}\+Normal"{}, "{}\+Eco"{}, "{}\+Intensive"{}, "{}\+Quick"{}, and "{}\+Rinse\+Only"{}, each with corresponding timer values.
