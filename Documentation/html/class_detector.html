Detailed Description

Represents a security detector that is linked to a SecurityAlarm.

Abstract class representing a security detector that is linked to a SecurityAlarm.

This class inherits from the SecurityDevice class and is associated with an instance of the SecurityAlarm class.

This class inherits from the SecurityDevice class and is associated with an instance of the SecurityAlarm. It serves as a base class for different types of security detectors that perform detection and trigger alarms.
