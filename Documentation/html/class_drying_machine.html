Detailed Description

Represents a DryingMachine device with various drying modes and corresponding timers.

Represents a drying machine that can operate in different modes.

The DryingMachine class inherits from HasMode and allows the user to select different drying modes such as "Normal", "Delicate", "Baby", "Quick", "Synthetic", and "AntiWrinkle". Each mode has a corresponding timer setting. The class provides methods to display information, set modes, and calculate energy consumption.

The DryingMachine class inherits from HasMode and provides functionality to set various drying modes. Each mode has a predefined time and affects the energy consumption and thread level of the machine.
