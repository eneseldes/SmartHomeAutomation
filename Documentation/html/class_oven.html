Detailed Description

A class representing an oven with different cooking modes and temperature control.

A class representing an oven with various modes and temperature control.

The Oven class is designed to simulate an oven with multiple operation modes such as Conventional, Fan, Grill, and Defrost. It allows setting and getting the oven's temperature, selecting the operation mode, and controlling energy consumption and thread levels.

This class provides functionality to set and get the temperature of the oven, select the operation mode (e.g., Conventional, Fan, Grill, Defrost), manage energy consumption, and perform reset operations.
