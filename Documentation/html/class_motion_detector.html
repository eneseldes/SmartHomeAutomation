Detailed Description

A class representing a motion detector that triggers an alarm when motion is detected.

The MotionDetector class is a specialized type of detector that checks for motion in a room. If motion is detected, it triggers the associated alarm. This class inherits from the Detector class and implements its abstract methods.

Detects motion in a room and triggers the associated alarm.

This method checks if the room has users and triggers the alarm if motion is detected. If no motion is detected, it reports that the room is safe.
