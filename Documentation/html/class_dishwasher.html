Detailed Description

Represents a Dishwasher device with adjustable modes and timer functionality.

The Dishwasher class inherits from HasMode and provides functionality to set and retrieve operation modes, display information about the device, and calculate energy consumption. It includes predefined modes like "Normal", "Eco", "Intensive", "Quick", and "RinseOnly", each with corresponding timer values.
